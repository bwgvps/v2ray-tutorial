- zz-v2rayN-With-Core-SelfContained.7z，包含所有Core文件；独立包，不需要下载安装[Microsoft .NET 6.0 Desktop Runtime]

由于GitHub限制文件大小，所以部分下载包这里无法提供，大家可以去官网查看：[v2rayN releases](https://github.com/2dust/v2rayN/releases)。
